# なにこれ？（Focus Translator）

右Ctrlキーを押すとマウスポインタ直下のテキストを認識して、翻訳をカーソル周辺にオーバーレイ表示するWindows常駐ツールです。
日本語にローカライズされていないソフトウェアやSaaSの管理画面のメニュー・ボタン・エラーメッセージなどの意味を即座に確認するために使用します。

![画面イメージ（SaaSメニューの翻訳）]()

## 開発経緯

業務で海外のSaaSやソフトウェアを使用する際、英語のメニューやボタン、エラーメッセージの意味がわからず、「テキストを選択してコピーし、ブラウザの翻訳ツールに貼り付ける」という作業を繰り返すことに煩わしさを感じていました。

そこで、「『なにこれ？』と思った瞬間に、マウスカーソルが指している箇所だけをピンポイントで翻訳・解説してくれるツール」として「なにこれ？（Focus Translator）」を開発しました。

![画面イメージ（エラーメッセージの解説）]()

右Ctrlキーを押している間だけサッと翻訳が表示され、離せばすぐに元の作業に戻れます。画像化されていてコピーできないテキストであってもOCRで瞬時に読み取れます。さらに「ただ翻訳するだけでなく、生成AI（LLM）の力でその文脈やIT専門用語の解説までしてくれる」ため、業務のフォーカス（Focus）を一切削ぐことのない、シームレスな翻訳体験を提供します。

## インストールと常駐化

本ツールはスタートアップ常駐を前提としています。以下の手順でインストールしてください。

1. [Releases]() から `focus-translator-setup.exe` をダウンロードします。
2. インストーラを実行し、画面の指示に従ってインストールします。
3. 自動的にスタートアップに登録され、タスクトレイに常駐します。

### インストーラのビルド (開発者向け)

Inno Setup Compiler (`ISCC.exe`) を使用して独自のインストーラをビルドできます。

```bat
cargo build --release
"C:\Program Files (x86)\Inno Setup 6\ISCC.exe" installer.iss
```

生成物: `Output\focus-translator-setup.exe`。

## 設定・モデルの保存先

既定では `%APPDATA%\FocusTranslator\` (config.json, models\, ログ) に保存されます。

## 使い方

| 操作                | 動作                                                              |
| ------------------- | ----------------------------------------------------------------- |
| **右Ctrl ホールド** | カーソル下の1行を認識・翻訳して表示。離すと消える                 |
| **Ctrl+Alt+T**      | 範囲指定モード。ドラッグ選択した矩形をOCR・翻訳(最初からピン留め) |
| **チップ押下**      | OCR/翻訳エンジンを切替えて再処理。押下時点でピン留め              |
| **コピー**          | 訳文(なければ原文)をクリップボードへ                              |
| **Esc / 閉じる**    | ピン留め表示を閉じる                                              |
| **タスクトレイ**    | 設定画面 / 範囲指定翻訳 / 終了                                    |

## 画面一覧と詳細解説

FocusTranslatorは、タスクトレイ常駐型アプリとして以下の主要なUI画面を備えています。

### 1. オーバーレイ画面（翻訳結果表示）

キャプ立直後にカーソル付近にポップアップするメイン画面です。

![画面イメージ（オーバーレイ画面）]()

- **基本構成**: 上段に小さく「認識した原文」、下段に大きく「翻訳結果」を表示します。
- **チップ（ボタン群）**: 結果の下に並ぶボタンで、エンジン切替、解説（LLMによるIT用語の詳細な意味や用途の解説）、キャプチャ編集、コピーなどの操作が可能です。
- **ピン留め（固定）機能**: ホールドキーを設定秒数長押しするか、ウィンドウ内をドラッグ操作することで、キーを離しても画面が残る「ピン留め」状態になります。

### 2. キャプチャ編集画面

![画面イメージ（キャプチャ編集）]()

オーバーレイ画面の【キャプチャ画像】ボタンから展開する、インラインの画像編集機能です。
複雑なUIから特定のテキストだけを投げ輪や矩形で切り抜き、OCRの認識精度を向上させます。

### 3. 設定画面

![画面イメージ（設定画面）]()

タスクトレイメニューから開く、総合設定画面です。

- **操作・OCR設定**: ホットキーや既定OCRエンジンの選択。
- **翻訳設定**: 既定翻訳エンジン、APIキー（DeepL / Google / Gemini等）の設定。
- **LLMプロファイル設定**: 業務ドメインに合わせた解説用プロンプトを個別に設定可能。

### 4. プロンプト編集とログビューアー画面

![画面イメージ（ログビューアー）]()

- **プロンプト編集**: `{{original_text}}` などの変数を活用し、LLMに送信する命令文を細かくカスタマイズできます。
- **ログビューアー**: 過去の業務中の翻訳履歴をツリー構造で振り返り、再処理やCSV出力による用語集の作成が行えます。履歴はローカルのSQLiteに安全に保存されます。

## エンジン対応状況

| エンジン                      | 種別         | 状態                                                                |
| ----------------------------- | ------------ | ------------------------------------------------------------------- |
| OneOCR (oneocr.dll)           | OCR (既定)   | ✅ 実装済み (Windows 11 Snipping Tool同梱・インストール不要)        |
| Windows.Media.Ocr             | OCR          | ✅ 実装済み                                                         |
| PaddleOCR                     | OCR          | ⏳ モデル導入(設定画面からワンクリックDL)は実装済み。推論は次版対応 |
| Gemini                        | OCR+翻訳統合 | ✅ 実装済み                                                         |
| DeepL / Google Trans / Gemini | 翻訳         | ✅ 実装済み                                                         |
| ローカルONNX翻訳              | 翻訳 (既定)  | ✅ 実装済み (FuguMT 等によるローカルオフライン翻訳)                 |

## プライバシーとセキュリティ

業務利用を想定し、セキュリティに最大限配慮しています。

- 既定構成 (Windows OCR + ローカル翻訳) では外部へのデータ送信は一切行われません。
- クラウドAPI（Gemini, DeepL等）を利用する際は、初回送信時に同意ダイアログを表示します。
- 登録されたAPIキー等の機密情報は、Windowsの DPAPI（Data Protection API）で暗号化してローカルに保存されます。ログへの出力はマスクされます。

## モデルの導入(ワンクリックインストール)

設定画面から、ローカルで完結する高精度なOCRや翻訳モデルをワンクリックで導入できます。いずれもダウンロード後にSHA256ハッシュを検証し、安全性を確保しています。

- **PaddleOCR**: より高精度なテキスト検出・認識を行います。
- **ローカルONNX翻訳**: 機密情報を含むテキストをオフラインで安全に翻訳するためのローカルモデル（FuguMT等）です。
